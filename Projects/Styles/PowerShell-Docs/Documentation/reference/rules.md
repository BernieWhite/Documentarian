## Avoid

Consider replacing "by using" with "using" - the leading "by" is redundant.

{{< vale/rule "Avoid" >}}

### Examples

Incorrect
: The cmdlet changes the file by using the passed parameters.

Correct
: The cmdlet changes the file using the passed parameters.

## Cliches
